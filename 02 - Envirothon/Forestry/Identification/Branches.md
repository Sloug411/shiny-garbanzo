#Study #Forestry 

There are three main branch configurations.

![[alternatebranch.png]]

**Alternate**: The Alternate Branch Configuration is when branches sprout on opposite sides staggered from each other. Most trees are alternate.


![[oppositebranch.png]]
**Opposite**: The Opposite Branch Configuration is when branches sprout off at the same place on opposite sides. In P.A. trees with this configuration are often either Maple, Ash, or Dogwood, which can be shortened to M.A.D.

![[whorledbranches.png]]
**Whorled**: The Whorled Branch Configuration is when branches sprout out in rings of branches. These are mainly [[Conifers]], with a few exceptions.
